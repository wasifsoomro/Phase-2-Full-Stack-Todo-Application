# TaskFlow - Modern Todo Application

A sleek, professional task management application with modern UI design and enhanced user experience to help you organize your life and manage tasks efficiently.

## Features

- **Modern Design**: Clean, professional interface with subtle gradients and depth effects
- **Responsive Layout**: Works seamlessly across all device sizes
- **Dark/Light Mode**: Automatic theme switching with consistent color schemes
- **Smooth Animations**: Polished transitions and micro-interactions throughout
- **Task Management**: Create, edit, and organize tasks efficiently
- **Status Tracking**: Visual indicators for task status (To Do, In Progress, Completed)

## Design Highlights

- **Color Palette**: Sophisticated blue, purple, and teal color scheme
- **Typography**: Clear visual hierarchy with modern font stack
- **Components**: Consistent design language across all UI elements
- **Accessibility**: WCAG AA compliant with proper contrast ratios
- **Performance**: Optimized animations and transitions for smooth experience

## Tech Stack

- Next.js 14+
- TypeScript
- Tailwind CSS
- shadcn/ui components
- Lucide React icons

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Customization

The application uses CSS variables for theming. You can customize the color scheme by modifying the variables in `app/globals.css`.
